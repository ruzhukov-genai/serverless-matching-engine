# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records — lightweight docs capturing significant technical decisions, their context, and rationale.

|-----|-------|--------|
| [ADR-001](ADR-001-serverless-architecture.md) | Adopt Serverless (Lambda) Architecture | Accepted |

## Format

Each ADR follows the pattern:
- **Status:** Proposed / Accepted / Deprecated / Superseded
- **Context:** Why does this decision need to be made?
- **Decision:** What was decided?
- **Consequences:** Trade-offs, risks, follow-ups
